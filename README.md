# Proyecto 4: Red de Hospitales y Atención Médica
## Eficiencia Clínica, Ingresos y Desempeño por Especialidad

## Archivos incluidos:
- pacientes.csv
- medicos.csv
- consultas.csv

## Instrucciones:
1. Cargue TODOS los archivos CSV en Power BI Desktop
2. Use Power Query para limpiar los errores documentados en el Word
3. Construya el modelo de datos resolviendo la relación Muchos a Muchos
4. Implemente las 5 medidas DAX
5. Construya el dashboard con mínimo 5 visualizaciones
6. Aplique principios de Data Storytelling

## Relación Muchos a Muchos:
- Tabla A: Pacientes
- Tabla B: Medicos  
- Tabla Puente: Consultas

## Preguntas de Negocio:
1. ¿Qué especialidad médica genera mayores ingresos y tiene mayor volumen de consultas?
2. ¿Cuál es la distribución de resultados de consulta (Alta, Tratamiento, Seguimiento, Cirugía) y cómo varía por hospital?
3. ¿Existen médicos con duración de consulta muy por encima del promedio (posible ineficiencia o casos complejos)?

## KPIs requeridos:
- Total Consultas
- Ingresos Médicos
- % Alta Médica
- Duración Promedio
- Especialidad Top
